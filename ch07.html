Types of Variables

1. User-defined — you create these
BACKUP_DIR="/u01/backups"
ALERT_EMAIL="dba@company.com"
MAX_RETRIES=3
2. Environment — set by the system/Oracle
echo $ORACLE_HOME    # Oracle install path
echo $ORACLE_SID     # Database name
echo $HOME           # Home directory
echo $USER           # Current username
echo $PATH           # Command search path

# Export makes a variable available to child processes
export ORACLE_SID="PRODDB"
3. Special — built into Bash
$?    # Exit code of last command (0=success)
$0    # Name of the current script
$#    # Number of arguments passed
$1    # First argument
$2    # Second argument
$$    # Process ID of current script

# Example: check if sqlplus succeeded
sqlplus -s / as sysdba @check.sql
if [ $? -ne 0 ]; then
    echo "SQL*Plus failed!"
fi
4. Positional — arguments from the command line
# When you run: ./myscript.sh PRODDB 85
$0  # → ./myscript.sh
$1  # → PRODDB
$2  # → 85

#!/bin/bash
DB_NAME=$1
if [ -z "$DB_NAME" ]; then
    echo "Usage: $0 <database_name>"
    exit 1
fi
Best practice: Define ALL variables at the top of your script. Never hard-code paths or values in the middle of the logic.
