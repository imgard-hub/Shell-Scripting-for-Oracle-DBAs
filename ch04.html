Your Tools

You don't need anything extra. Everything is already on the server:

  • A text editor: vi/vim, nano (easier for beginners), or VS Code locally
  • A terminal: SSH into your database server
  • bash: already installed on every Linux system
  • chmod: to make scripts executable
  • cron: to schedule scripts (built into Linux)
Essential commands
vi myscript.sh           # Edit in vi
nano myscript.sh         # Edit in nano (easier)
chmod +x myscript.sh     # Make executable
./myscript.sh            # Run it
bash -x myscript.sh      # Debug mode (shows each step)
Verify your Oracle environment
echo $ORACLE_HOME     # Should show Oracle install path
echo $ORACLE_SID      # Should show database name
echo $PATH            # Should include $ORACLE_HOME/bin

# If blank, set them:
export ORACLE_HOME=/u01/app/oracle/product/19c
export ORACLE_SID=ORCL
export PATH=$ORACLE_HOME/bin:$PATH
Editor tip for beginners: Use nano — it shows shortcuts at the bottom. Type nano myscript.sh. Switch to vi later once comfortable.
