Types of Shells

There are several shell programs available on Linux/Unix. Each interprets commands slightly differently:

ShellNamePathNotes
bashBourne Again Shell/bin/bashMost common. Default on most Linux. This course uses bash.
shBourne Shell/bin/shThe original. More limited. Often symlinked to bash.
kshKorn Shell/bin/kshCommon on older Solaris/AIX. Very similar to bash.
csh/tcshC Shell/bin/cshC-like syntax. Rarely used for DBA scripts.
zshZ Shell/bin/zshModern, powerful. Default on macOS. Compatible with most bash.
How to check your shell
# What shell are you using right now?
echo $SHELL
# Output: /bin/bash  (this is what you want)

# What shells are available?
cat /etc/shells

# Check the oracle user's shell
grep oracle /etc/passwd
Bottom line: Use bash. It's the standard, it's everywhere, and every example in this course uses bash.
